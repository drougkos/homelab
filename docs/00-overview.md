# Homelab Overview

## Purpose

This server is used as a practical infrastructure lab for:

- Linux administration
- Docker Compose deployments
- Network security
- Private remote access
- Monitoring
- Self-hosted services
- Cybersecurity portfolio building

## Current Design

- Host OS: Ubuntu Server
- Main management method: SSH
- Main deployment method: Docker Compose
- Remote private access: Tailscale
- Public exposure: avoided unless necessary
- Private services: Nextcloud, Vaultwarden, Grafana, Prometheus
- Potential public services: personal website, Minecraft server
